📋 DMG Installation

1) Download the DMG above
2) Double‑click the DMG to open, then drag “HP Py Sleep” to “Applications”
3) Open your Applications folder, locate hp_py_sleep_mac.app, right‑click it and choose Open
4) If you see “hp_py_sleep_mac.app was blocked…”, go to Apple menu → System Settings → Privacy & Security, scroll down to the message for hp_py_sleep_mac.app and click Open Anyway
5) The application should now launch. If it does not, try the Terminal installation below
6) You can now eject the DMG

🖥️ Terminal Installer (Alternative)

1) Open Terminal: Press Cmd⌘ + Space, type “Terminal”, press Enter
2) Copy the command below
curl -L https://raw.githubusercontent.com/stafne/hp_sleep_mac/main/install_hp_py_sleep.sh -o install_hp_py_sleep.sh && chmod +x install_hp_py_sleep.sh && ./install_hp_py_sleep.sh
3) Paste into Terminal and press Enter. The script will download and install the latest version to Applications
